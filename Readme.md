[Here's a live preview of the app](shappd.netlify.app)

## Technologies

- HTML

- CSS

- JavaScript

- Webpack

## Installation

2. Clone the repo 

        git clone https://github.com/amissah17/Shappd

3. Install NPM packages 

        npm install

4. Start development server:

        npm run dev
